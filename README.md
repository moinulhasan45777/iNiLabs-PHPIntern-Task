## Description
This repository is a combination of 6 assignments assigned by iNiLabs Recruitment Team in order to evaluate my knowledge and skills.

## Task 0
**Task Name: Valid Parentheses**

This task is a problem solving task. Here I was told to ensure that in the input string all the bracktes are closed in the correct sequence or order. 

## Task 1
**Task Name: Class Inheritance**

In this task I was instructed to implement and showcase Class Inheritance with classes representing geometric shapes including circle and rectangles. Also, I was told to calculate the area with methods.

## Task 2
**Task Name: Interface Implementation**

Here I had to create a logging interface to develop a logging system by making classes implement the logging interface. In other words, forcing the classes to use the methods declared inside the interface.

## Task 3
**Task Name: Encapsulation**

This task is built to showcase Encapsulation by designing a class to manage employee data (employee salary) and using setters and getters to securely set and get employee salaries.

## Task 4
**Task Name: Polymorphism**

Here I showcased Polymorphism by developing a set of animal classes which has overriden a method named 'makeSound()' declared in an abstract parent class 'Animal' to make sounds of each animal separately.

## Task 5
**Task Name: To-Do Module**

Here the Laravel Framework has been used to create the backend API's integrating CORS middleware. Furthermore, Vue.js has been used to create the frontend part of the task integrating Axios to handle the API requests easily and efficiently. MySql RDBMS has been used to create and manage the database and entities.
